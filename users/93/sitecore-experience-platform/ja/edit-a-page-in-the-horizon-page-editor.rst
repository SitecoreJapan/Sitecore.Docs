###################################################
Horizon ページエディタでページを編集する
###################################################

Horizonページエディタは、Webサイトのコンテンツをページ上で直接変更できるWYSIWYGエディタです。テキスト、グラフィック、ロゴ、リンクなど、ページに表示されているすべての項目を編集することができます。

Horizon エディタでは、ページの最終バージョンを編集しているので、変更内容は以下のすべてに影響します。

* ページの最終レイアウト
* すべての言語のすべてのバージョンで共有されている共有フィールド
* 同じ言語のすべてのバージョンで共有されているフィールドのうち、逆コンパイルされていないもの
* 現在の言語のアイテムの現在のバージョンにのみ適用されるバージョン付きフィールド

.. note:: Horizon エディタでフィールドを編集する場合、フィールドに関連するコンテンツが含まれている場合、同じコンテンツを使用している他のページのコンテンツも変更されることに注意してください。

Horizon ページ エディタでページを編集するには

1. Horizon エディタにログインして、編集するページを開きます。
2. 異なる言語バージョンに切り替えるには、アプリケーション URL の言語パラメータを変更します。たとえば、URL に sc_lang=en が含まれている場合、英語版からフランス語版に切り替えるために en を fr に置き換えることができます。
3. 編集したいフィールドをクリックして、内容を入力します。編集するフィールドのタイプによって、異なるオプションがあります。

  * テキストフィールド (1 行または複数行): テキストのみを入力できます。入力したテキストに合わせてフィールドが自動的に展開されます。
  * :doc:`リッチテキストフィールド <edit-a-rich-text-field-in-the-horizon-editor>` フィールド内のテキストを編集したり、右側のペインに表示される書式設定ツールを使用してテキストに書式設定を適用することができます。
  * :doc:`画像フィールド <edit-an-image-field-in-horizon>` さまざまな種類の画像を挿入できます。

4. デバイススイッチャー |icon1| でデバイスの種類をクリックすると、別のデバイスでどのようなページが表示されているかを確認することができます。

.. |icon1| image:: images/15eafd3519c675.png

.. note:: どのデバイスビューで行った変更も、すべてのデバイスに影響します。サイトに実装されているレスポンシブ動作によっては、異なるデバイスでレンダリングが移動したり、表示されたり、消えたりする場合があります。
