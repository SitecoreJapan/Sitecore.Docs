##########################################
エンタイトルメントポリシー
##########################################

Entitlement プラグインは、プラグインが提供する機能の動作を定義する設定ポリシーを提供します。

.. note::

    すぐに使えるポリシーを拡張したり、継承したりしないでください。

    ポリシーが定義されている場所に応じて、PolicySet JSONまたはEnvironment JSONを修正して、Bootstrapを再実行することで、デフォルトのポリシーの値を変更することができます。

Entitlement プラグインは、以下のポリシーを定義します。

+--------------------------------+----------------------------------------------------------+---------------------------------------------------------------------------------------+
| ポリシー                       | 説明                                                     | プロパティ / 標準値」                                                                 |
+--------------------------------+----------------------------------------------------------+---------------------------------------------------------------------------------------+
| KnownEntitlementsActionsPolicy | エンタイトルメントビューのアクションの名前を定義します。 | DeleteEntitlement \ "DeleteEntitlement"                                               |
+--------------------------------+----------------------------------------------------------+---------------------------------------------------------------------------------------+
| KnownEntitlementsViewsPolicy   | エンタイトルメント・ビューの名前を定義します。           | OrderEntitlements \ "OrderEntitlements" CustomerEntitlements \ "CustomerEntitlements" |
|                                |                                                          | OrderEntitlementDetails \ "OrderEntitlementDetails"                                   |
|                                |                                                          | CustomerEntitlementDetails \ "CustomerEntitlementDetails"                             |
+--------------------------------+----------------------------------------------------------+---------------------------------------------------------------------------------------+
