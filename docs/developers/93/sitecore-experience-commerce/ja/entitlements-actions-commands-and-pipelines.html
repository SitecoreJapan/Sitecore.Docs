エンタイトルメント アクション、コマンド、パイプライン¶

以下の表では、エンタイトルメントのアクション、コマンド、およびパイプラインについて説明します。

Action/Command/Pipeline

説明

Get]Api/Entitlements

FindEntitiesInListCommand

FindEntitiesInListPipeline

システム内のすべてのエンタイトルメントを返します。 1

Get]Api/Entitlements(id)

FindEntityCommand

FindEntityPipeline

識別子に基づいた特定のエンタイトルメントを返します。

以下のパラメータを持っています。

Id - エンタイトルメントの識別子。

DeleteEntitlement

DeleteEntitlementCommand

DeleteEntitlementPipeline

エンタイトルメントを削除します。
1

これは計算量が多いので、注意して使用する必要があります。
