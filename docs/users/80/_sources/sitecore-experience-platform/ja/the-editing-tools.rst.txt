####################################
編集ツール
####################################


Sitecore には Web サイト上のコンテンツを作成、編集できる編集ツールが 2 つあります。

* コンテンツ エディター - Sitecore と Sitecore に含まれる機能に精通している経験豊富なコンテンツ編集者向けに設計されたアプリケーションです。
* エクスペリエンス エディター - 直接ページ上からコンテンツを編集したり書き込んだりするコンテンツ編集者向けに設計された編集環境です。

Web サイトを構成するアイテムはコンテンツ エディターとエクスペリエンス エディターでまったく異なった方法で表示されます。コンテンツ エディターでは、アイテムはコンテンツ ツリー内のオブジェクトとなり、アイテムを選択するとそのフィールドを編集することができます。エクスペリエンス エディターでは、アイテムは Web サイト上と同じように表示され、直接ページ上から編集することができます。

例えば、コンテンツ エディターの場合、サンプル Web サイトの Services アイテムは次のように表示されます：

.. image:: images/blank.png
   :align: center
   :width: 400px
   :alt: 空白

コンテンツ ツリーで Service アイテムをクリックすると、[コンテンツ] タブに Service アイテムの編集可能なフィールド値が表示されます。他のアイテムを編集したい場合はコンテンツ ツリーからアイテムをクリックする必要があります。

エクスペリエンス エディターの場合、Web サイトで表示するのと同じように Service アイテムは表示されます。

.. image:: images/blank.png
   :align: center
   :width: 400px
   :alt: 空白

例えば、テキスト、図表、ロゴ、リンクなど、たとえ選択されたアイテムに属していない場合でも、ページ上のすべてのアイテムを編集することができます。
