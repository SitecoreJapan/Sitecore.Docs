Sitecore カスタマー エクスペリエンス マネージメント ソフトウェアには、顧客の素晴らしい Web サイト体験を自由自在に操る最先端の CMS およびマーケティング機能が導入されています。これによって、キャンペーンから訪問者アクティビティ、パフォーマンス評価に至るあらゆるすべてのアクティビティを複数のチャネル上で統一することができます。
