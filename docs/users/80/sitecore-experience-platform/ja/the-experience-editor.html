エクスペリエンス エディター¶

エクスペリエンス エディターは Sitecore が提供する最もシンプルな編集ツールです。エクスペリエンス エディターは、直接ページ上から簡単にアイテムを変更できる WYSIWYG エディターです。ページ上に表示されるすべてのアイテム (例えば、テキスト、図表、ロゴ、リンクなど) を編集することができます。

エクスペリエンス エディターでは、選択中のアイテムのフィールドをだけでなく、ページにレンダリングされている任意のアイテムのフィールドも編集できます。例えば、ページのメニューが 製品アイテムのタイトルフィールドで構成されている場合、製品 アイテムに移動しなくてもそのタイトルを編集することができます。

リボン¶

エクスペリエンス エディターでは、有効なすべての機能を備えたリボンが Web ページの上部に表示されます。リボンの機能は、セキュリティー ロールの設定と Web サイトのカスタマイズ状態によって異なります。

エクスペリエンス エディターのすべての機能への完全なアクセスがある場合は、それぞれ固有のコマンドを備えた 5 つのタブにアクセスすることができます。

最低限でも、Web サイト上のページの挿入と削除は実行可能なことが想定されます。どのようなリボンを使用していたとしても、必ずリボンの左上には [保存] ボタンがあります。

アイテムを編集する際は、[リボンの切り替え] ボタンをクリックして一時的にリボンを非表示にし、編集スペースを広げることができます。

リボンを表示するには、再度 [リボンの切り替え] ボタン をクリックします。
