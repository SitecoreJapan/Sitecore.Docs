電子メール リマインダーを設定する¶

Sitecore では、電子メール リマインダーを 1 人以上のユーザーに送信するようスケジューリングすることができます。この機能は、Web サイトに時間的制約のあるコンテンツがある場合に役立ちます。例えば、Web サイトに特定の日付で終了する特別オファーがある場合、適切な日に Web サイトからそのオファーを削除するように自分や他のユーザーにリマインダーを送信することができます。

リマインダーを設定するには、次の手順を実施します：

  1. コンテンツ エディターで、[レビュー] タブの [スケジュール] グループで [リマインダー] をクリックします。オプションの一覧から [リマインダーを設定] をクリックします。

  1. [Set Reminder] ウィザードの [Welcome to the Set Reminder Wizard] ページで [Next] をクリックします。

  2. [電子メールのリマインダーを送信する] ダイアログ ボックスの [リマインダー テキスト] フィールドで、受信者にリマインダーとして送信したいテキストを入力して [次へ] をクリックします。

  1. [日付と受信者] ダイアログで、[日時] フィールドに電子メール リマインダーを送信したい日付と時刻を入力します。[受信者] フィールドに受信者の電子メール アドレスを入力して [設定] をクリックします。

  1. [閉じる] をクリックします。

リマインダーの送信日を確認するには、[レビュー] タブの [スケジュール] グループで [リマインダー] をクリックします。リマインダーに関する詳細は [リマインダー設定] オプションの上部に表示されます。

設定したリマインダーを削除するには、[レビュー] タブの [スケジュール] グループで [リマインダー] をクリックして、[リマインダーをクリア] をクリックします。

注釈

アイテムごとに設定できるリマインダーは 1 つのみです。既にリマインダーが設定されているアイテムにリマインダーを設定したい場合は、先に設定されているリマインダーを上書きします。
