編集ツール¶

Sitecoreでは、Webサイトのコンテンツを作成・編集できる編集ツールが2つあります。

  • The Content Editor - Sitecoreとその機能を熟知している、より経験豊富なコンテンツ制作者向けのアプリケーションです。

  • The Experience Editor - ページ上で直接コンテンツを編集・執筆したいコンテンツ編集者のために設計された、直感的な編集環境です。

コンテンツエディタとエクスペリエンスエディタでは、ウェブサイトを構成するアイテムの表示方法が大きく異なります。コンテンツエディタでは、アイテムはコンテンツツリー内のオブジェクトであり、アイテムを選択すると、そのフィールドを編集することができます。エクスペリエンス エディタでは、項目はウェブサイト上に表示され、ページ上で直接編集することができます。

例えば、コンテンツ エディタでは、サンプル ウェブサイトのサービスの項目は次のように表示されます。

コンテンツ ツリーで サービス の項目をクリックすると、コンテンツ タブにサービス項目のフィールド値が表示されます。別の項目を編集する場合は、コンテンツ ツリーでその項目をクリックする必要があります。

エクスペリエンス エディタでは、サービス アイテムが Web サイトに表示されます。

選択した項目に属していなくても、テキスト、グラフィック、ロゴ、リンクなど、ページ上のすべての項目を編集することができます。
