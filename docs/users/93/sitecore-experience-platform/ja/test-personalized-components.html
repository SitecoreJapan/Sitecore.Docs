パーソナライズされたコンポーネントのテスト¶

対象となるすべてのサイト訪問者に対してパーソナライズを有効にする前に、パーソナライズされたコンポーネントのテストを実行するとよいでしょう。パーソナライゼーションのテストを期間限定で実行し、テストに割り当てられる Web サイト トラフィックの量を選択します。

テストが終了し、対象となるすべてのサイト訪問者にパーソナライゼーションを適用した後、 パーソナライゼーション効果のトラッキング を使用して、パーソナライゼーションのパフォーマンスを継続的に監視することができます。

パーソナライゼーションテストを開始¶

  1. コンポーネントにパーソナライズルールを設定します

  2. [コンポーネントのパーソナライズ］ダイアログボックスで、［このコンポーネントのパーソナライゼーションの効果を追跡する］チェックボックスを無効にして、［OK］をクリックします。

  3. 変更を保存します。

  4. アイテムがワークフローの一部である場合は、そのアイテムをワークフローに通します。

通知バーにテストを開始するための通知が表示されます。これで、 プレビューしてテストを開始 できます。

重要

コンポーネントにパーソナライゼーション ルールを設定すると、デフォルトでは、そのコンポーネントに対してパーソナライゼーション効果の追跡が有効になります。コンポーネントのパーソナライゼーション テストを作成するには、このコンポーネントのパーソナライゼーションの効果を追跡するオプションを無効にする必要があります。
