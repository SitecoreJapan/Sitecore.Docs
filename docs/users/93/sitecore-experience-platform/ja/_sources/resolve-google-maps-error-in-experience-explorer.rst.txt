#################################################################################
エクスペリエンス エクスプローラで Google マップのエラーを解決する
#################################################################################

エクスペリエンスエディタでは、ExploreモードのビューアパネルにGoogleマップを表示して訪問者の位置を表示することができます。しかし、Google Maps API が変更されると、ビューアパネルには Google マップの代わりにエラーメッセージが表示されます。

Google マップのエラーを解決するには

1. ブラウザで、次のページを開きます:
   https://developers.google.com/maps/documentation/javascript/get-api-key。
2. ページの指示に従って、API キーを取得します。
3. コンテンツ エディタで、/sitecore/system/Modules/Experience Explorer/Map Settings/Google のアイテムで、API キーフィールドにキーを入力します。
4. ブラウザのキャッシュをクリアします。

.. note:: キャッシュのクリア方法がわからない場合は、ブラウザのヘルプを参照してください。

これで地図が正しく表示されるようになりました。

.. tip:: 英語版 https://doc.sitecore.com/users/93/sitecore-experience-platform/en/resolve-google-maps-error-in-experience-explorer.html
