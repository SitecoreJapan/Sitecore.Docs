###################################
バージョンがクローンに与える影響
###################################

複数のバージョンを持つアイテムのクローンを作成したい場合や、複数のバージョンのクローンを作成したい場合は、クローンの更新方法を定義する特定のルールを知っておく必要があります。


+--------------------------------------------------+------------------------------------------------------------------------------------------------------------------+
| アクション                                       |                                                                                                                  |
+==================================================+==================================================================================================================+
| 複数のバージョンを持つアイテムをクローンする     | クローンは、クローンを作成したときに選択したオリジナルのアイテムのバージョンにリンクします。                     |
|                                                  | クローンを作成する前に、元のアイテムの適切なバージョンを選択してください。                                       |
+--------------------------------------------------+------------------------------------------------------------------------------------------------------------------+
| オリジナルアイテムに新バージョンを追加           | クローンに通知が表示され、対応する新しいバージョンのクローンを作成するかどうかを尋ねてきます。                   |
+--------------------------------------------------+------------------------------------------------------------------------------------------------------------------+
| 新しいバージョンのクローンの追加                 | 新しいバージョンは、前のバージョンのクローンがリンクしていた元のアイテムのバージョンにリンクしています。         |
+--------------------------------------------------+------------------------------------------------------------------------------------------------------------------+
| 元のアイテムのバージョンを更新する               | 元のアイテムのその特定のバージョンにリンクされているクローン版のみが、それに応じて更新されます。                 |
+--------------------------------------------------+------------------------------------------------------------------------------------------------------------------+
| クローンを持つ元のアイテムのバージョンを削除する | 元のアイテムのその特定のバージョンにリンクされているクローン版はすべてアンクローン化されています。               |
|                                                  | これは、これらのバージョンのクローンが、元のアイテムから値を継承しない通常のアイテムに変わったことを意味します。 |
+--------------------------------------------------+------------------------------------------------------------------------------------------------------------------+
