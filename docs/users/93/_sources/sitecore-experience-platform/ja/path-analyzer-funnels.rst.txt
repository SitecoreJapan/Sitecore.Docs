######################################################
Path アナライザーのファンネル
######################################################

ファネルとは、ビジネスの目的を達成するために顧客がウェブサイト上で行うことが期待される、明確に定義された一連のステップのことです。独自のファネルを作成することができます。eコマースサイトでは、最も分かりやすいのはチェックアウトファネルで、ウェブサイトのデザインによっては、ページ、ページイベント、ゴールのシーケンスとして表現することができます。

Path Analyzerでは、ファネルを使用することで、顧客が目標や成果に変換するまでの行動や経路を分析することができます。パーソナライゼーション、M/Vテスト、コンテンツテストを使用して、これらのパスを最適化し、コンバージョン率を向上させることができます。

ファネルを選択すると、ファネルの追跡ステップを棒グラフでグラフィカルに表示することができます。チャートでは、ファネルの各ステップを完了した顧客の数や、次のステージに進む顧客の数などの情報が得られます。また、訪問数、価値、訪問当たりの価値などの主要な指標を切り替えることができます。チャートの下には、他の全てのメトリクスのリストが表示され、ファネルの主要なパフォーマンス指標を提供します。これらのメトリクスは、ファネルの最後のステップから取得されます。このデータは、生涯のカスタマージャーニーとは対照的に、1回のインタラクションの範囲に基づいて計算されます。

パスファネルは、何人のコンタクトがパスから抜け出し、どのノードから抜け出したかを示しています。パスファネルから、ファネルレポートにアクセスできます。これは、このパスを通過した全てのコンタクト体験の価値、訪問数、訪問当たりの価値を表示します。

ファネルをお気に入りにマークすると、ダッシュボードに表示されます。
